# 🚀 Digital Agency Website (Practice Project)

A modern and high-performance Digital Agency website built using **React JS**, **Tailwind CSS**, and **Framer Motion**. This project serves as a complete hands-on guide for building, integrating, and deploying a production-ready web application.

---

## 📌 Overview

This project demonstrates the full workflow of creating a professional agency-style website — from frontend design to live deployment. It focuses on performance, modern UI/UX, and real-world features.

---

## 🎯 Key Features

### 🎨 Modern UI/UX

* Light/Dark mode toggle
* Custom animated cursor
* Scroll-triggered animations
* Clean and professional layout
* Fully responsive design

### ⚛️ Frontend Development

* Built with **React JS**
* Styled using **Tailwind CSS**
* Smooth animations using **Framer Motion**

### 📩 Contact Form Integration

* Powered by **Web3Forms**
* No backend required
* Sends messages directly to email

---

## 🛠️ Tech Stack

* React JS
* Tailwind CSS
* Framer Motion
* Web3Forms
* Vite

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/digital-agency-project.git
cd digital-agency-project
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Project

```bash
npm run dev
```

---

## 🔑 Environment Setup

Create a `.env` file and add:

```env
VITE_WEB3FORMS_ACCESS_KEY=your_access_key_here
```

Get your key from: https://web3forms.com
